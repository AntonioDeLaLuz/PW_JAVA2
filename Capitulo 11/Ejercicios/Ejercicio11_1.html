11.1 Complete las siguientes oraciones

a) Como javascript usa objetos para realizar muchas tareas, se conoce por lo comun como lenguajes de programacion basado en objetos

b) Todos los objetos tienen atributos y exhiben comportamientos

c) Los métodos del objeto Math nos permiten realizar muchos calculos matematicos comunes.

d) Al proceso de invocar(o llamar) al metodo de un objeto se le conoce como enviar un mensaje al objeto

e) Las literales de cadena o constantes de cadena se escriben como una secuencia de caracteres en comillas dobles o comillas sencillas

f) Los indices para los caracteres en una cadena empieza en 0

g) Los métodos indexOf y lastIndexOf de string buscan la primera y ultima ocurrencia de una subcadena en un objeto string, respectivamente.

h) Al proceso de dividir una cadena en tokens se le denomina tokenizacion

i) El Procesamiento de fechas y horas puede realizarse con base en la zona horaria local de la computadora o en el tiempo universal coordinado (UTC) del estandar de tiempo mundial

j) El metodo parse de Date recibe como argumento una cadena que representa una fecha y hora, y devuelbe el numero de milisegundos trascurridos entre la medianoche del 1 de enero de 1970 y la fecha y hora especificada.

k) Las aplicaciones web pueden usar la propiedad localStorage del objeto window para almacenar hasta varios megabytes de datos de cadenas de pares clave/valor en la computadora del usuario y pueden acceder a estos datos a traves de sesiones y pestañas del navegador.

l) Las aplicaciones web que necesitan acceso a los datos de pares clave/valor para sólo una sesión de navegacion y que deben mantener esos datos separados entre varias pestañas pueden usar la propiedad sessionStorage del objeto window.

m) Un sector :first-child de CSS3 selecciona el primer hijo de un elemento

n) Un sector nth-child de CSS3 con el argumento "odd" selecciona los elementos hijos impares y uno con el argumento "even" selecciona los eleemntos hijos pares.
